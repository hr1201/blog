# nestcli
 
## 初始化
1. 创建项目：
```shell
npm i -g @nestjs/cli

nest new [项目名称]
```

2. 可以使用`nest --help`查看nest的常用命令：

```shell
# 生成controller.ts
nest g co user

# 生成  module.ts
nest g mo user

# 生成service.ts
nest g s user

# 一键生成
nest g resource user
```
